Installation

Get started with Friday AI in minutes

1

Install with pip

Install the Friday AI package from PyPI

pip install friday-ai-teammate
2

Configure API Keys

Set your LLM provider API key and base URL

export API_KEY=your-api-key
export BASE_URL=https://api.openai.com/v1
# Or use a .env file
3

Run Friday AI

Start interactive mode or run a single command

# Interactive mode
friday
# Single prompt
friday "explain this codebase"
# Autonomous mode
friday
> /autonomous 50
4

Optional: Setup .claude Folder

Create a .claude folder for project-specific configuration

mkdir -p .claude/{agents,skills,rules,commands,workflows}
# Friday will automatically load resources from here
